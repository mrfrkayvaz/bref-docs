# Tools

Maybe one day, the Bref format will be natively supported by programming languages. Until that day comes, you can parse Bref strings and perform JSON conversions using the tools below.

### NodeJS

```js
import fs from 'fs';
import bref from 'bref';

const content = bref.parse(data);
console.log(content);
```
