# Tools

Maybe one day, the Bref format will be natively supported by programming languages. Until that day comes, you can rely on Bref converters to seamlessly transform your data into JSON on the fly.

### Python
[Python Bref Extension](https://github.com/mrfrkayvaz/bref)

```python copy
import bref

with open("data.bref", "r", encoding="utf-8") as f:
  content = bref.toJSON(f.read())

print(content)
```
